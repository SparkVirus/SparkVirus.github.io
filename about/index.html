About me

Hi, my name is Shem Kiptoo. I am currently a fourth year student pursuing BSc. Computer Science at Kabarak University in Nakuru, Kenya. I enjoy coding and solving programming problems. I also like sharing what I know with my peers so we can learn from each other and develop together.

What I know

I am quite conversant with web development(I prefer backend though) and android app development. I have taken part in some programming contests using C++ and Java as my most preferred languages, though I have been using Python for my projects lately :-).

I have built most of my web development projects using PHP, specificaly Laravel and CodeIgniter frameworks, for server side scripts, and jQuery and Angular javascript frameworks for the frontends.

My philosophy: research, experiment, fail, google, fix, fail again, stackoverflow, fix… At the end of the day you will have learnt something that can take you a long way. College syllabus and books are not enough.

As a developer I believe that code implements a solution and it’s not the solution itself. So if you’re looking to figure out how good I am in a given language then I suggest that you give me a problem (with a good motivation to work on the problem) and specify the language you want me to solve the problem in. If I can’t I will let you know. I have not taken any standardized proficiency tests for any language, framework or any sort of technology and therefore, I cannot specify how good I am at any one of them.
